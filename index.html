sTabs jQuery plugin by Smasty

Tiny plugin which makes tabs feature from simple unordered lists.

API

These options are available for the plugin:
int startWith
default: 1
Number of tab to activate as first.
string eventType
default: 'click'
Event to handle changing tabs. Must be a valid jQuery.bind() event type.
bool animate
default: false
Whether or not animate changing tabs.
int duration
default: 300
Duration of animation in miliseconds.

Options should be passed to plugin as an object.

Demo

Default functionality

$(function(){
  $('#default').sTabs();
});
